// 04 - Règles d'usage

Do. Don't.

Do - palette

  • Un seul accent hot par vue principale
  • Couleurs plates, jamais de gradient
  • Vérifier le contraste WCAG AA avant de valider
  • Les CTAs (boutons primaires, install blocks) restent sombres en dark mode pour préserver les accents bleu et yellow

Don't - palette

  • Pas de hex hors de la liste sans mise à jour de PALETTE.md
  • Pas de violet (cramé par l'AI slop 2023-2024)
  • Pas de rouge signal pour les actions principales (rouge = erreur)
  • Pas de yellow comme couleur d'action - c'est un highlight, pas un CTA

Do - typographie

  • Trois registres maximum sur une page : Archivo Narrow, Fraunces italic, JetBrains Mono
  • Italic uniquement pour les accents courts (taglines)
  • Letter-spacing négatif sur les gros titres condensés
  • Letter-spacing positif (0.16-0.2em) sur les labels uppercase

Don't - typographie

  • Pas d'italique sur du body texte multi-lignes (illisibilité dyslexie)
  • Pas de mono dans les paragraphes
  • Pas de 4e font "pour varier"
  • Pas d'em-dash (-) jamais

Do - voix

  • Direct, factuel, pas de marketing
  • Imperatives sur les CTAs ("Install once", "Open project")
  • Lowercase pour les labels système
  • Build with cc en trailer de commit

Don't - voix

  • Pas de "the future of...", "revolutionary", "10x faster"
  • Pas de stats sans source ni de citations sans auteur
  • Pas de décoration creuse type "vol. 01" ou "chapter 02"
  • Pas de stock photos, pas d'illustrations isométriques
  • Pas d'emoji dans le UI

// 05 - Anti-patterns visuels

à ne jamais reproduire
  • Watermarks géants en filigrane. Mangent l'espace utile, dégradent le contraste, sentent le mood-board AI.
  • Hero qui prend 75% du viewport. Le pattern "above the fold" pour un site dev = titre court + 1.5 cards visibles.
  • Gradients violets indigo. Signal universel "encore un wrapper GPT" depuis 2023.
  • Toggle sun/moon décoratifs. Préférer un toggle auto / light / dark textuel.
  • Numérotation décorative. "Vol. 01", "Chapter II" - bruit cognitif sans information.
